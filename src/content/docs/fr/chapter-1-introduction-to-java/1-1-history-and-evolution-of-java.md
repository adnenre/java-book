Java a été créé par **James Gosling**, Mike Sheridan et Patrick Naughton chez **Sun Microsystems** en 1991. Initialement nommé **Oak** (d’après un chêne devant le bureau de Gosling), il était conçu pour la télévision interactive et les systèmes embarqués. Le projet était trop avancé pour l’industrie de la télévision par câble à l’époque.

En 1995, le langage a été renommé **Java** (en référence au café Java) et réorienté vers le World Wide Web naissant. La première version publique, **Java 1.0**, promettait “Write Once, Run Anywhere” (WORA) grâce à la Java Virtual Machine (JVM).

**Étapes clés** :

- **1996** – Java 1.0 (première version officielle)
- **1997** – Java 1.1 (classes internes, JavaBeans, JDBC)
- **1999** – Java 2 (J2SE 1.2) avec Swing, le framework Collections
- **2004** – Java 5 (génériques, annotations, enum, boucle for améliorée)
- **2009** – Oracle acquiert Sun Microsystems
- **2014** – Java 8 (expressions lambda, Stream API, API date/heure)
- **2017** – Java 9 (modules, JShell)
- **2018** – Java 11 (LTS, première version Oracle avec nouvelle licence)
- **2021** – Java 17 (LTS, classes scellées, pattern matching)
- **2023** – Java 21 (LTS, threads virtuels, patterns de records)
- **2025** – Java 24 (Stream Gatherers, API Class-File, cryptographie résistante aux quantiques)
- **2025** – Java 25 (LTS, dernière version de support à long terme, en-têtes d’objets compacts, améliorations AOT)
- **2026** – Java 26 (client HTTP/3, améliorations du débit G1 GC, suppression définitive de l’API Applet)

Java a évolué d’un langage simple et indépendant de la plateforme vers un écosystème robuste qui alimente les applications d’entreprise, mobiles et cloud. Avec le rythme de publication semestriel établi en 2017, Java continue d’innover rapidement tout en maintenant la rétrocompatibilité.
