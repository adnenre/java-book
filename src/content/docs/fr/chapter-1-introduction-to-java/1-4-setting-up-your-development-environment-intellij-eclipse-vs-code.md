# Configuration de l’environnement de développement

Un bon environnement de développement intégré (IDE) rend le développement Java productif. Voici trois choix populaires.

## IntelliJ IDEA

- **Édition** : Community (gratuite) ou Ultimate (payante).
- **Fonctionnalités** : excellente complétion de code, refactoring, contrôle de version intégré, support des frameworks.
- **Configuration** :
  1. Téléchargez depuis [jetbrains.com/idea](https://www.jetbrains.com/idea/).
  2. Installez et lancez.
  3. Créez un nouveau projet – sélectionnez Java et indiquez l’emplacement du JDK.
  4. Écrivez le code, exécutez avec le bouton vert.

## Eclipse

- **Édition** : gratuit et open‑source.
- **Fonctionnalités** : très extensible, adapté aux grands projets, vaste écosystème de plugins.
- **Configuration** :
  1. Téléchargez depuis [eclipse.org](https://www.eclipse.org/downloads/).
  2. Décompressez et lancez `eclipse.exe`.
  3. Choisissez un dossier d’espace de travail.
  4. Créez un nouveau projet Java, configurez le JRE si nécessaire.

## Visual Studio Code

- **Édition** : gratuit, léger.
- **Fonctionnalités** : rapide, avec des extensions riches.
- **Configuration** :
  1. Téléchargez depuis [code.visualstudio.com](https://code.visualstudio.com/).
  2. Installez le **“Extension Pack for Java”** (inclut le support du langage Red Hat, débogueur, etc.).
  3. Ouvrez un dossier, créez un fichier `.java` et commencez à coder.
  4. Utilisez le terminal intégré pour compiler/exécuter ou cliquez sur le bouton “Run”.

Les trois IDE prennent en charge le débogage, la navigation dans le code et l’intégration avec des outils de build (Maven, Gradle).
